El catálogo se guarda en este navegador con localStorage. Para las emisiones de YouTube, lo más estable es abrir este HTML desde un servidor local y no con file://. En Windows puedes abrir PowerShell en la carpeta del archivo y ejecutar:
python -m http.server 8000
Después abre http://localhost:8000/worldcam_minimal_v0_4.html.
