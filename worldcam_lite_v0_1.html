Añadir cámara manualmente

Guarda cámaras públicas propias en este navegador. No se sube nada a ningún servidor.

Importar / exportar

Formato JSON simple. Útil para pasar tu catálogo a otro PC o ir ampliando la base de datos poco a poco.

La clave mínima es: title, country, lat, lon, type y una fuente compatible.

Arquitectura de esta primera versión

Esta versión es un único archivo HTML: estilos internos, lógica JavaScript interna y catálogo inicial incrustado. No necesita PHP para funcionar.

PHP o Node serían útiles más adelante si quieres buscar miles de webcams automáticamente, normalizar datos, cachear APIs o usar un proxy legal para flujos con CORS.

Limitaciones reales del navegador

No todas las webcams se pueden incrustar. Algunos proveedores bloquean iframes, otros solo dejan imágenes periódicas y otros exigen clave API.

La app prioriza YouTube embebido, iframes permitidos, imágenes públicas y enlaces originales como respaldo.

Privacidad y uso correcto

Usa solo cámaras públicas, oficiales o autorizadas. No está pensado para cámaras privadas, RTSP doméstico, paneles vulnerables ni streams encontrados por escaneo.

Modo día / noche

El tema se guarda automáticamente en este navegador. Puedes cambiarlo con el botón de la cabecera.
