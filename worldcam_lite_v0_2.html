Añadir cámara

Tipos admitidos: iframe embebible, YouTube, imagen refrescada y HLS básico. Para YouTube pega solo el ID o la URL completa.

Importar / exportar

El catálogo añadido se guarda en localStorage de este navegador.

Consejo práctico: para evitar el Error 153 de YouTube, esta versión añade referrerpolicy="strict-origin-when-cross-origin", usa youtube-nocookie.com y añade origin cuando el archivo se sirve por http://localhost. Si se abre como file://, algunos reproductores pueden seguir bloqueándose por no recibir Referer real.
